Meu Primeiro Portal Spring Boot

Padrão MVC - Model View Controller

* Model: Faz o processamento e validação de dados do sistema.

* View: Fornece a organização visual do sistema, apresentando os dados do Model e captando a interação do usuário.

* Controller: Recebe os inputs do usuário, se comunicando com o Model para alterar ou resgatar dados e definir qual View deve ser exibida.
