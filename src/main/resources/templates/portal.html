Meu Primeiro Portal Spring Boot

Padrão MVC - Model View Controller

* Model: Classes base para contrução do abjeto.

* View: Fornece a organização visual do sistema, apresentando os dados do Model e captando a interação do usuário.

* Controller: Gerencia o sitema, controlando o fluxo de dados.
